南山人壽實健溢百住院醫療終身保險：計畫30

假設情境 :
王小明，75歲，因膝關節半月軟骨損傷住院10天，其中3天入住加護病房。住院期間進行了膝關節半月軟骨修補術，並因心臟問題植入一個心臟支架。病房費差額每天5000元，雜費手術費37萬，看護費每天3000元。出院後2周內門診2次，每次800元，超過2周也門診2次，每次800元。
二、理賠分析
1. 每日病房費用保險金：50,000元
2. 住院/門診手術費用保險金 (雜費及手術費)：370,000元
3. 住院前後門診費用保險金：1,000元
總理賠金額：421,000元
理賠排除告知：
1. 看護費：每日3,000元，此險種不提供看護費用給付。
2. 出院後超過2周門診費用：2次，每次800元，此費用已超出保單約定之給付期間（出院後2周內），故不予給付。
三、各理賠項目計算式

1. 每日病房費用保險金

理賠分析：

1. 被保險人王小明為75歲，適用74歲(含)後實支實付型醫療保障。
2. 王小明住院10天，其中3天入住加護病房，每日病房費差額為5,000元。
3. 根據保單條款，若曾入住加護病房，計畫30的每日病房費用限額為6,000元。
4. 每日病房費用保險金給付金額為實際支出與限額兩者取較小者。

計算式：

1. 每日病房費用保險金 = 住院天數 X 每日病房費差額
2. 每日病房費用保險金 = 10天 X 5,000元 = 50,000元
3. 因實際支出5,000元小於每日限額6,000元，故給付50,000元。

2. 住院/門診手術費用保險金 (雜費及手術費)

理賠分析：

1. 被保險人王小明為75歲，適用74歲(含)後實支實付型醫療保障。
2. 王小明住院期間進行膝關節半月軟骨修補術並植入心臟支架，雜費手術費共37萬元。
3. 根據保單條款，計畫30的雜費及手術費限額為60萬元。
4. 雜費及手術費保險金給付金額為實際支出與限額兩者取較小者。

計算式：

1. 雜費及手術費保險金 = 實際雜費手術費
2. 雜費及手術費保險金 = 370,000元
3. 因實際支出370,000元小於限額600,000元，故給付370,000元。

3. 住院前後門診費用保險金

理賠分析：

1. 被保險人王小明為75歲，適用74歲(含)後實支實付型醫療保障。
2. 王小明出院後2周內門診2次，每次費用800元。
3. 根據保單條款，74歲(含)後住院前後門診費用採實支實付，每次限額500元，且僅給付住院前2周及出院後2周的門診，每天限給付1次。
4. 出院後超過2周的門診不予給付。

計算式：

1. 住院前後門診費用保險金 = 出院後2周內門診次數 X 每次門診費用限額
2. 住院前後門診費用保險金 = 2次 X 500元 = 1,000元
3. 因每次實際支出800元大於每次限額500元，故以限額計算。
4. 超過2周的門診不符合給付條件，不予計算。
