南山人壽實健溢百住院醫療終身保險：計畫30

假設情境 :
王小明，65歲，因膝關節半月軟骨損傷住院10天，其中3天入住加護病房。住院期間進行了膝關節半月軟骨修補術，並因心臟問題植入一個心臟支架。
二、理賠分析
1. 住院日額保險金：30,000元
2. 加護病房暨燒燙傷中心加額：9,000元
3. 住院手術醫療保險金：4,000元
4. 特殊醫材補助保險金：15,000元
總理賠金額：58,000元
三、各理賠項目計算式
理賠分析
1. 住院日額保險金：被保險人王小明住院10天，符合住院日額保險金給付條件。
2. 加護病房暨燒燙傷中心加額：被保險人王小明住院期間有3天入住加護病房，符合加護病房暨燒燙傷中心加額給付條件。
3. 住院手術醫療保險金：被保險人王小明進行膝關節半月軟骨修補術，符合住院手術醫療保險金給付條件。
4. 特殊醫材補助保險金：被保險人王小明因心臟問題植入心臟支架，符合特殊醫材補助保險金給付條件。
計算式
1. 住院日額保險金：每日3,000元 X 住院10天 = 30,000元
2. 加護病房暨燒燙傷中心加額：每日3,000元 X 加護病房3天 = 9,000元
3. 住院手術醫療保險金：膝關節半月軟骨修補術，定額給付4,000元
4. 特殊醫材補助保險金：心臟支架，每次15,000元 X 1次 = 15,000元
總理賠金額：30,000元 + 9,000元 + 4,000元 + 15,000元 = 58,000元
